# GoArgentina

Projeto simples para ajudar brasileiros no processo de obtenção do DNI argentino.

## Estrutura

- `index.html` – Landing page de vendas (foco em conversão para WhatsApp).
- `dashboard.html` – Painel do cliente para envio de documentos e acompanhamento do processo de DNI.
- `admin.html` – Backoffice administrativo para gestão de clientes, documentos e fases do processo.
